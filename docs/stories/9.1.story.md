# Story 9.1: Integrar autenticacao cloud por conta no companion web

**Epic:** EPIC-007 (AudioFlow Web Sync)  
**Status:** In Progress  
**Priority:** COULD  
**Effort:** M (Medium)  
**Phase:** 9 - Cloud Sync & Hybrid Companion  

---

## Story

**Como** usuario macOS do companion web  
**Eu quero** entrar com minha propria conta cloud no companion  
**Para que** eu acesse meus dados com identidade propria sem perder o fallback local ja entregue  

---

## Acceptance Criteria

- [ ] O companion web oferece signup e login por email/senha com Supabase Auth no modo cloud
- [ ] O frontend persiste a sessao cloud e restaura o usuario autenticado apos reload sem depender da senha fixa local
- [ ] O backend Node valida JWT por requisicao com `SUPABASE_ANON_KEY` + `Authorization: Bearer <token>` e nao usa service role em runtime
- [ ] Rotas cloud retornam erro controlado quando o token esta ausente ou invalido, sem expor dados de outro usuario
- [ ] O modo local atual continua acessivel como fallback explicito quando o cloud nao estiver habilitado ou disponivel
