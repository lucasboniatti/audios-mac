## Technical Details

### Key APIs

- `NSPasteboard.general`

### Files to Create

- `Services/ClipboardService.swift`

### ClipboardService.swift

```swift
import Cocoa

class ClipboardService {
    static let shared = ClipboardService()
    
    func copyToClipboard(_ text: String) {
        let pasteboard = NSPasteboard.general
        pasteboard.clearContents()
        pasteboard.setString(text, forType: .string)
        
        // Post notification
        NotificationCenter.default.post(name: .textCopied, object: nil)
    }
    
    func getClipboardText() -> String? {
        return NSPasteboard.general.string(forType: .string)
    }
}

extension Notification.Name {
    static let textCopied = Notification.Name("textCopied")
}
```

---

## Integration

```swift
// No AppDelegate ou Controller
speechService.onFinalResult = { [weak self] text in
    ClipboardService.shared.copyToClipboard(text)
    self?.showNotification("Texto copiado para o clipboard")
}
```

---

## Dev Notes

- Usar `clearContents()` antes de `setString()` para garantir clipboard limpo
- Notificação pode ser um simples alert ou NSUserNotification
