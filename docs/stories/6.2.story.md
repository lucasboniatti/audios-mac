## Technical Details

### Performance Test

```swift
func testStartupLatency() {
    measure {
        let start = Date()
        HotkeyController.shared.simulateToggle()
        let elapsed = Date().timeIntervalSince(start)
        XCTAssertLessThan(elapsed, 0.5) // < 500ms
    }
}
```
