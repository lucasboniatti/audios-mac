## Technical Details

### Key APIs

- `AVAudioEngine`
- `AVAudioInputNode`
- `AVAudioSession` (não usado no macOS, apenas iOS)

### Files to Create

- `Controllers/AudioController.swift`

### AudioController.swift

```swift
import AVFoundation

class AudioController {
    private var audioEngine: AVAudioEngine?
    private var inputNode: AVAudioInputNode?
    
    func startRecording() throws {
        audioEngine = AVAudioEngine()
        inputNode = audioEngine?.inputNode
        
        let recordingFormat = inputNode?.outputFormat(forBus: 0)
        
        inputNode?.installTap(onBus: 0, bufferSize: 1024, format: recordingFormat) { buffer, time in
            // Buffer será enviado para SpeechService
            NotificationCenter.default.post(
                name: .audioBufferAvailable,
                object: nil,
                userInfo: ["buffer": buffer]
            )
        }
        
        try audioEngine?.start()
    }
    
    func stopRecording() {
        inputNode?.removeTap(onBus: 0)
        audioEngine?.stop()
        audioEngine = nil
    }
}
```

### Notification Extension

```swift
extension Notification.Name {
    static let audioBufferAvailable = Notification.Name("audioBufferAvailable")
}
```

## Error Handling

| Error | Behavior |
|-------|----------|
| Microfone indisponível | Mostrar erro no menu, não crash |
| Permissão negada | Alert com instrução |
| Dispositivo mudou | Re-inicializar audio engine |

---

## Dev Notes

- Usar `installTap(onBus:)` para capturar buffer de áudio
- Buffer é enviado via NotificationCenter para SpeechService
- **NÃO salvar buffer em arquivo** - apenas passar para transcrição
