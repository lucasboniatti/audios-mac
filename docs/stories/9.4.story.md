# Story 9.4: Implementar dashboard cloud com filtros por periodo

**Epic:** EPIC-007 (AudioFlow Web Sync)  
**Status:** Draft  
**Priority:** COULD  
**Effort:** M (Medium)  
**Phase:** 9 - Cloud Sync & Hybrid Companion  

---

## Story

**Como** usuario autenticado do dashboard web  
**Eu quero** filtrar minhas metricas e listagens por periodo no modo cloud  
**Para que** eu analise meu uso por data sem perder o fallback local quando a sincronizacao falhar  

---

## Acceptance Criteria

- [ ] O dashboard cloud aplica filtros por periodo (`today`, `7days`, `month` e intervalo customizado) nas APIs de listagem e estatisticas
- [ ] As consultas do dashboard respeitam o usuario autenticado e validam datas/intervalos antes de executar a query
- [ ] Metricas e listagens permanecem consistentes quando o filtro muda ou a pagina recarrega
- [ ] Quando a nuvem nao responde, o dashboard degrada para leitura local controlada com indicacao clara de fallback
- [ ] O comportamento existente do dashboard local nao regride para usuarios sem modo cloud habilitado
