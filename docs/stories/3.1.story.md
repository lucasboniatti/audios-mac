## Technical Details

### Approach

Usar `RegisterEventHotKey` para registrar `Control+A` como hotkey global do sistema.

**NOTA:** Hotkey registrado evita depender de event tap e reduz conflito visual com o app em foco.

### Files to Create

- `Controllers/HotkeyController.swift`

### HotkeyController.swift

```swift
import Cocoa

class HotkeyController {
    private var hotKeyRef: EventHotKeyRef?
    private var onToggle: (() -> Void)?
    
    init(onToggle: @escaping () -> Void) {
        self.onToggle = onToggle
    }
    
    func start() {
        RegisterEventHotKey(...)
    }
    
    func stop() {
        if let hotKeyRef = hotKeyRef {
            UnregisterEventHotKey(hotKeyRef)
            self.hotKeyRef = nil
        }
    }
    
    private func showRegistrationAlert() {
        let alert = NSAlert()
        alert.messageText = "Atalho Indisponível"
        alert.informativeText = "Nao foi possivel registrar o atalho global Control+A. Esse atalho pode estar reservado pelo sistema ou em uso por outro app."
        alert.alertStyle = .warning
        alert.addButton(withTitle: "OK")
        alert.runModal()
    }
}
```

## Error Handling

| Error | Behavior |
|-------|----------|
| Falha ao registrar atalho global | Alert informando conflito com sistema/outro app |
| Conflito com outro app | Log warning, não bloquear |
| App terminando | Parar monitor antes |
