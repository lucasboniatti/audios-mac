## Technical Details

### Approach

Usar `NSEvent.addGlobalMonitorForEvents(matching: .keyDown)` para detectar Space globalmente.

**NOTA:** Hotkeys globais requerem permissão de Accessibility no macOS.

### Files to Create

- `Controllers/HotkeyController.swift`

### HotkeyController.swift

```swift
import Cocoa

class HotkeyController {
    private var eventMonitor: Any?
    private var onToggle: (() -> Void)?
    
    init(onToggle: @escaping () -> Void) {
        self.onToggle = onToggle
    }
    
    func start() {
        // Check Accessibility permission
        let options = [kAXTrustedCheckOptionPrompt.takeUnretainedValue() as String: true]
        let trusted = AXIsProcessTrusted()
        
        if !trusted {
            showAccessibilityAlert()
            return
        }
        
        eventMonitor = NSEvent.addGlobalMonitorForEvents(matching: .keyDown) { [weak self] event in
            // Space key code = 49
            if event.keyCode == 49 {
                self?.onToggle?()
            }
        }
    }
    
    func stop() {
        if let monitor = eventMonitor {
            NSEvent.removeMonitor(monitor)
            eventMonitor = nil
        }
    }
    
    private func showAccessibilityAlert() {
        let alert = NSAlert()
        alert.messageText = "Permissão Necessária"
        alert.informativeText = "AudioFlow precisa de permissão de Accessibility para detectar o hotkey global.\n\nVá em System Preferences > Privacy & Security > Accessibility e adicione AudioFlow."
        alert.alertStyle = .warning
        alert.addButton(withTitle: "Abrir Preferências")
        alert.addButton(withTitle: "Cancelar")
        
        if alert.runModal() == .alertFirstButtonReturn {
            NSWorkspace.shared.open(URL(string: "x-apple.systempreferences:com.apple.preference.security?Privacy_Accessibility")!)
        }
    }
}
```

### Key Code Reference

| Key | KeyCode |
|-----|---------|
| Space | 49 |
| Return | 36 |
| Escape | 53 |

---

## Error Handling

| Error | Behavior |
|-------|----------|
| Sem permissão Accessibility | Alert com instrução para System Preferences |
| Conflito com outro app | Log warning, não bloquear |
| App terminando | Parar monitor antes |

## Dev Notes

- **NÃO usar Carbon API** (deprecated)

- Fallback: Cmd+Shift+Space se Space causar conflitos com outros apps
- Testar com diferentes apps em foco (Terminal, Browser, etc.)
