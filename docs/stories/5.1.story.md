## Technical Details

### Export Format

```
AudioFlow - Histórico de Transcrições
Exportado em: 2026-03-06 10:30:00
----------------------------------------

[2026-03-05 14:30:00] Primeira transcrição aqui...

[2026-03-05 14:35:00] Segunda transcrição aqui...
```

### Implementation

- `HistoryController.exportToTXT(to:)` - Exporta todas as transcrições
- Menu item "Exportar TXT..." no submenu Histórico
- Save panel com filtro .txt
- Notificação de sucesso/erro
